## 🕹️ How to Play

### 🎯 Rock Paper Scissors
-- **How to Play:** Choose **rock, paper, or scissors** and compete against the computer.

### 🎯 Guess the Number
--**How to Play:** **Guess the correct number** and see if it matches the randomly generated number.

### 🎯 General Knowledge Quiz
--**How to Play:** Answer a series of questions and see **how many you can get right**.

### 🎯 Hangman Game
--**How to Play:** **Guess the hidden word** within a limited number of attempts.

### 🎯 Tic Tac Toe
--**How to Play:** **Make strategic moves** to get **three in a row** and win the game.

### 🎯 Anagram Solver
--**How to Play:** Input a set of letters, and the program will find all **possible anagrams**.

## 🛠️ Getting Started

### Requirements
- Python 3.6 or higher
- Terminal/Command Prompt

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/GRISHMA-GEDELA/GameHub-Python.git
   cd GameHub-Python
   ```

2. Run any game directly. For example:
   ```bash
   python "Rock_Paper_Scissors.py"
   # or
   python "Tic_Tac_Toe.py"
   ```

### Game Files
- `Rock_Paper_Scissors.py` - Classic rock-paper-scissors game
- `Guess_a_Number.py` - Number guessing game
- `GK_Quiz.py` - General Knowledge quiz game
- `Hangman_Game.py` - Word guessing game
- `Tic_Tac_Toe.py` - Classic tic-tac-toe game
- `Anagram_Solver.py` - Find word anagrams
- `All the Code using Modules.py` - Combined version of all games
